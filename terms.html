许可方信息

本设计系统的许可方为 CGArtLab（以下简称「我们」或「许可方」）。许可方对本设计系统及其内容享有著作权或其他相关知识产权（适用法律允许的范围内）。

如需联系许可方，请访问 CGArtLab.com 或通过 GitHub 仓库。

开源许可证

CGArtLab 设计系统采用 Creative Commons 署名 4.0 国际许可证（CC BY 4.0） 发布。这意味着您可以：

  • 自由使用 — 在任何项目中采用本设计系统，无论个人或商业目的
  • 自由复制 — 将本系统的全部或部分内容复制到您的项目中
  • 自由修改 — 根据您的需求修改、扩展或定制本设计系统
  • 自由分发 — 在遵守许可证条款的前提下，向他人分享或分发本系统
CC BY 4.0 官方链接

本系统采用 Creative Commons Attribution 4.0 International Public License（CC BY 4.0）。如需查看完整法律文本，请访问 CC BY 4.0 法律代码（英文）。

您享有的权利

在遵守本许可证条款的前提下，您可以：

  • 在商业产品、服务或内部工具中使用本设计系统
  • 修改设计令牌的数值（如颜色、间距、字体等）以适应您的品牌
  • 将修改后的版本用于闭源或开源项目
  • 基于本系统创建并销售衍生产品（如 UI 组件库、主题包）
  • 在不受信任或受限的环境中部署使用

本许可证不要求您支付任何费用或获得额外授权。

闭源使用说明

CC BY 4.0 允许将衍生作品用于闭源项目。您只需确保原始设计系统部分仍可依 CC BY 4.0 使用，并保留适当的署名。

署名要求

使用本设计系统时，您必须：

  1. 保留版权声明 — 在您分发的作品副本中保留原始版权声明和本许可证的链接
  2. 注明来源 — 提供指向本设计系统的链接（如 designsystem.cgartlab.com 或 GitHub 仓库）
  3. 注明是否修改 — 如果您对本系统进行了修改，请在您的作品中注明「已基于 CGArtLab 设计系统修改」或类似表述
署名示例

「此项目使用 CGArtLab Design System（CC BY 4.0）」

商业使用

允许商业使用。 您可以将本设计系统用于商业产品或服务，包括但不限于：

  • 客户端项目（网站、应用、软件）
  • 内部工具与企业软件
  • SaaS 产品与订阅服务
  • 付费主题、模板或组件库

商业使用无需支付许可费用，但必须遵守本许可证中的署名要求。

修改与衍生

您可以对本设计系统进行任何形式的修改：

  • 换肤定制 — 修改颜色令牌以适配您的品牌色系
  • 组件扩展 — 在现有组件基础上添加新组件或变体
  • 代码重构 — 将 CSS 迁移到 SCSS、LESS、PostCSS 或其他预处理/构建工具
  • 框架适配 — 将样式迁移到 React、Vue、Svelte 等组件框架

根据 CC BY 4.0，衍生作品可以采用您选择的任何许可证。您对修改部分的贡献享有完整的知识产权，但必须确保：

  • 保留原始署名 — 在分发时保留对原始 CGArtLab 设计系统的适当署名
  • 提供许可证副本 — 随作品提供 CC BY 4.0 许可证副本
  • 不损害接收者权利 — 不采取措施妨碍接收者行使 CC BY 4.0 赋予的权利
关于衍生作品许可证

CC BY 4.0 不要求衍生作品保持 CC BY 4.0 许可证。您可以将修改后的版本用于闭源项目或采用其他许可证（如 MIT、Apache 2.0），只需确保原始设计系统部分仍可依 CC BY 4.0 使用。

在遵守本许可证条款的前提下，以下行为是不允许的：

CC BY 4.0 许可证不提供专利许可，也不限制任何人对本系统相关技术申请专利。专利问题由专利法另行规制。本许可证也不限制您对衍生作品采用其他许可证（如闭源或其他开源许可证）。
