# 行为准则（Code of Conduct）

## 我们的承诺

为了营造一个开放、友好的环境，我们作为贡献者和维护者承诺：不论年龄、体型、残疾、族裔、性别认同与表达、经验水平、国籍、个人形象、种族、宗教或性取向如何，参与本项目的每个人都享有无骚扰的体验。

## 我们的准则

有助于创造积极环境的行为包括：

- 🤝 使用欢迎与包容的语言
- 🎯 专注于对社区最有利的事情
- 💚 尊重不同的观点与经验
- 🧡 优雅地接受建设性批评
- 🌟 关注社区里他人的最佳利益
- 💜 对其他社区成员表示同理心

不可接受的行为包括：

- ❌ 性化的语言或图像，以及不受欢迎的性关注或性挑逗
- ❌ 侮辱/贬损的评论、人身或政治攻击
- ❌ 公开或私下的骚扰
- ❌ 未经明确许可，发布他人的私人信息（如物理地址或电子邮箱）
- ❌ 在专业场合中可被合理视为不当的其他行为

## 我们的责任

项目维护者有权利和义务去厘清、标记与拒绝任何不可接受的行为。
项目维护者有义务在他们认为合适的情况下，公开或私下回应不当行为。

## 适用范围

本行为准则适用于本项目的所有空间（GitHub Issues / PRs / Discussions / 邮件 / 线下聚会），
以及当个人代表本项目时的公开空间。

## 设计系统专用补充

由于本项目是**设计系统**而非普通软件项目，我们额外约定：

- 🎨 **视觉评审请聚焦于设计本身** — 评论实现方式而非审美偏好
- 🌍 **跨文化友好** — 色彩、排版、术语需考虑中文 / 英文 / 其他语言用户
- ♿ **可访问性优先** — 任何对 a11y 不利的改动需要明确说明并附替代方案
- 📚 **文档即代码** — 文档更新与代码更新同等重要，不要忽视

违反这些原则的评论或 PR 将由维护者根据严重程度处理：警告 → 关闭 PR → 临时封禁。
